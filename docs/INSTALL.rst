c189.theme Installation
-----------------------

To install c189.theme using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``c189.theme`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        c189.theme

* Re-run buildout, e.g. with:

    $ ./bin/buildout
